    Erstellen eines Videoclips mit Brightcove Social

    In diesem Thema erfahren Sie, wie Sie mit Brightcove Social einen Videoclip erstellen.

    Brightcove Social bietet die Möglichkeit, Videoclips aus vorhandenen Videos in Ihrem Video Cloud-Konto zu erstellen. Es gibt mehrere Gründe, warum Sie einen Clip erstellen möchten:

    • Um einen kurzen Teaser-Clip zu erstellen, der in sozialen Medien gepostet werden kann
    • So begrenzen Sie die Länge eines Videos, das Sie posten möchten (die meisten Plattformen begrenzen die Länge von Videos)
    • Sie haben festgestellt, dass kürzere Videos in den sozialen Medien besser abschneiden

    Erstellen eines Clips

    Nachdem ein Clip gespeichert wurde, wird er als Video in Ihrem Video Cloud-Konto gespeichert. Befolgen Sie diese Schritte, um einen Clip zu erstellen.
    1. Öffnen Sie Brightcove Social.
    2. Klicken Sie im linken Navigationsbereich auf Videos. Eine Liste aller Videos in Ihrem Video Cloud-Konto wird angezeigt.
    3. Klicken Sie auf das Video, um einen Clip zu erstellen, und klicken Sie dann auf Clip erstellen. Der Clip-Editor wird geöffnet.
      • Um einen Clip zu erstellen, muss dem Quellvideo mindestens eine MP4-Wiedergabeversion zur Verfügung stehen. Wenn die Schaltfläche „ Clip erstellen “ deaktiviert ist, steht dem Video möglicherweise keine MP4-Formatvariante zur Verfügung.
      • Sie können keinen Clip aus einem anderen Clip erstellen. Wenn Sie ein abgeschnittenes Video ausgewählt haben, wird die Schaltfläche „ Clip erstellen “ deaktiviert.
      • Sie können keinen Clip aus einem Video erstellen, für das DRM aktiviert ist (die Sperre wird im Videominiaturbild angezeigt) und mit einem Legacy-Ingest-Profil transcodiert wurde. DRM-fähige Videos, die mit einem Dynamic Delivery Ingest-Profil transcodiert wurden, können abgeschnitten werden. Wenn ein Clip nicht erstellt werden kann, ist die Schaltfläche „ Clip erstellen “ deaktiviert.
      • Videos mit Untertiteln können über Social abgeschnitten werden, das resultierende abgeschnittene Video wird jedoch keine Untertitel enthalten. Sie müssen neue Untertitel in das abgeschnittene Video hochladen, und Social verwendet diese bei der Veröffentlichung.
    4. Geben Sie einen Clip-Namenein.
    5. Ziehen Sie die auf der Timeline angezeigten Griffe, um den Anfang und das Ende Ihres Videos zu kürzen. Sie können auch bestimmte Start- und Endzeiten über die Bearbeitungsfelder eingeben. Eine genauere Steuerung beim Bearbeiten kann über die Tastenkombinationenerreicht werden. Wenn Sie fertig sind, klicken Sie auf Weiter.
      Hinweis: Der Spieler im Clipping-Modal scrubbt sich in ein oder zwei Frames zum angezeigten Zeitpunkt des tatsächlichen Frames. Das Verhalten ist das gleiche, unabhängig davon, ob Sie die Schieberegler mit der Maus ziehen oder die Tastenkombinationen verwenden, um die Clip-Punkte festzulegen. Der Clip-Editor wird genau zu den vorgesehenen Zeiten geklippt.
    6. Überprüfen Sie die Clip-Informationen. Der Clip-Name, die Tags und alle benutzerdefinierten Felder, die für das Konto erstellt wurden, können bearbeitet werden.
      Hinweis: Möglicherweise möchten Sie Tags entfernen, die dazu führen, dass der Clip auf Ihren vorhandenen Wiedergabelisten erscheint oder automatisch mit einem sozialen Ziel synchronisiert wird.
    7. Wenn Sie fertig sind, wählen Sie eine Option aus:
      • Speichern - Startet die Transkodierung und speichert den Clip dann in Ihrem Video Cloud-Konto.
      • Speichern & veröffentlichen - Startet die Transkodierung und speichert den Clip dann in Ihrem Video Cloud-Konto. Sie können auch das/die Ziel auswählen, in dem das Video veröffentlicht werden soll. Es kann zu einer Verzögerung kommen, dass veröffentlichte Clips auf Ihren Social-Media-Seiten angezeigt werden, da das Video transcodiert werden muss.
      • Abbrechen - Verwirft den Clip.

    Clips, die gespeichert werden, werden mithilfe des Aufgabe-Profils transcodiert, das als Standardwert für Social Clipping für das Konto festgelegt ist. Informationen zum Festlegen von Standard-Ingest-Profilen finden Sie unter Ingest-Profile verwalten.

    Beachten Sie, dass beim Erstellen von Clips das gesamte Quellvideo heruntergeladen werden muss. Bei längeren Quellvideos kann es länger dauern, bis der Clip in Ihrem Video Cloud-Konto verfügbar ist.

    Hinweis: Der Clip-Quelldateiname wird auf den gleichen Quelldateinamen wie das Quellvideo festgelegt.

    Verwenden von Tastenkombinationen

    Eine genauere Kontrolle über das Clipping kann mit den Tastenkombinationen erfolgen:

    Verknüpfung Beschreibung
    J oder ← Ein Frame zurück
    K Abspielen/Pause
    L oder → Ein Frame vorwärts
    I Markieren Sie den aktuellen Frame als Punkt
    O Markieren des aktuellen Frames als Out-Point
    Leertaste Abspielen/Pause
    + ICH Gehe zu In-Punkt
    + O Gehe zum Out-Punkt
