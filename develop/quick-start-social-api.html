    Kurzanleitung: Soziale API

    Dieses Thema bietet einen schnellen Einstieg in die Verwendung der Social API.

    Einführung

    Die Social API ermöglicht es Ihnen, den Status und die Geschichte des Teilens von Videos auf Social-Media-Plattformen zu erhalten. Mit diesem Schnellstart können Sie die verfügbaren Anfragen und ihre Antworten entweder mit Ihrem eigenen Konto oder einem Brightcove Learning Services-Beispielkonto anzeigen.

    Hinweise zur Beispiel-App

    • Dies ist ein schneller Einstieg, der Ihnen ein Modell zum Erstellen einer App rund um die Social API geben soll. Es ermöglicht Ihnen, grundlegende Anforderungen zu testen, verwendet jedoch keinen der Parameter, die diese Anforderungen verwenden können. Um diese zu sehen, sehen Sie sich die API-Referenzan.
    • Die App stellt auch einige Anfragen an den, CMS API um die Namen der Videos zu erhalten, die von der ersten Anfrage nach dem Status des sozialen Video-Sharings zurückgegeben wurden. Dies dient nur dazu, die Videoauswahl für die anderen Anfragen zu füllen, da die Social-API nur Video-IDs zurückgibt, nicht ihre Namen.
    Dieses Beispiel und der zugehörige Code dienen als Leitfaden für Ihre Produktionsentwicklung. Der Code ist weder für den produktiven Einsatz zertifiziert noch offiziell unterstützt.

    Anmeldedaten abrufen

    Um die Social API verwenden zu können, benötigen Sie die richtigen Anmeldeinformationen.

    Der einfachste Weg, sie zu erhalten, besteht darin, den Abschnitt Studio-Admin/API-Authentifizierung zu verwenden, auf Neue Anwendung registrieren zu klicken und die Berechtigungen für soziale Medien für Ihre Anmeldeinformationen auszuwählen:

    Schnellstart-App

    Finden Sie den gesamten mit diesem Beispiel verknüpften Code in diesem GitHub-Repository.

    Siehe den Schnellstart der Stift-Social API von Brightcove Learning Services (@rcrooks1969 ) auf CodePen.

    Verwenden des CodePen

    Hier sind einige Tipps, um das obige CodePen effektiv zu nutzen:

    • Schalten Sie die tatsächliche Anzeige des Players ein, indem Sie auf die Schaltfläche Ergebnis klicken.
    • Klicken Sie auf die Tasten HTML/CSS/JS , um EINEN der Codetypen anzuzeigen.
    • Klicke auf CodePen in der oberen rechten Ecke auf Bearbeiten , um diesen CodePen in dein eigenes Konto einzufügen.
    • Finden Sie den gesamten mit diesem Beispiel verknüpften Code in diesem GitHub-Repository.

    Proxy-Code

    Um Ihre eigene Version der Beispiel-App auf dieser Seite zu erstellen, müssen Sie Ihren eigenen Proxy erstellen und hosten. (Die von Brightcove Learning Services verwendeten Proxys akzeptieren nur Anfragen von Brightcove-Domänen.) Ein Beispiel-Proxy, der dem von uns verwendeten sehr ähnlich ist, aber ohne die Prüfungen, die Anfragen von Nicht-BrightCove-Domänen blockieren, kann in diesem GitHub-Repositorygefunden werden. Dort finden Sie auch grundlegende Anweisungen zur Verwendung und eine ausführlichere Anleitung zum Erstellen von Apps rund um den Proxy unter Using the REST APIs.
